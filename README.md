Kurdish Flashcards is a simple and effective language-learning application built with Flutter. It helps users memorize Kurdish vocabulary by presenting flashcards with Kurdish words and their English translations. The app is designed to support learners in expanding their vocabulary, practicing recall, and building confidence in everyday Kurdish usage.

Key Features:

📚 Flashcards with Kurdish words and English translations

🎯 Focused on memorization and vocabulary building

🔄 Easy navigation and smooth Flutter interface

🌍 Ideal for beginners and language enthusiasts
